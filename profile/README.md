# üîê Security & Privacy Philosophy

We follow **Local-First AI** principles:

- No cloud processing  
- No third-party analytics  
- No external data sharing  
- All ML inference is done on device  
- Open, explainable algorithms  
- Strict contributor confidentiality  

Security is enforced through:
- NDA + Contributor Licensing  
- CODEOWNERS  
- Private repos  
- Audit logs  
- 2FA for all members  
- GitHub branch protection  
- Zero external forks  

See `documentation/SECURITY_CHECKLIST.md` for full details.
